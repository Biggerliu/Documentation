:mod:`StepperMotor` --- 四相步进电机关联功能函数
==================================================

.. module:: Stepper motor
   :synopsis: 四相步进电机关联功能函数

``stepmotor`` 模块的主要功能与函数

功能相关函数
----------------------

.. only:: port_tpyboard
    
    .. function:: stepmotor.stepping(a,b,c,d,dir,speed)

    步进电机旋转函数。
        - ``a`` 驱动板IN1连接的引脚编号
        - ``b`` 驱动板IN2连接的引脚编号
        - ``c`` 驱动板IN3连接的引脚编号
        - ``d`` 驱动板IN4连接的引脚编号
        - ``dir`` 旋转的方向（1：顺时针，0：逆时针）
        - ``speed`` 旋转速度，最小值为2

四相步进电机与开发板接线对应引脚：
------------------------------------

		+------------+-----------+
		| TPYBoard   | 步进电机  |
		+============+===========+
		| VIN        |   ＋      |
		+------------+-----------+
		| GND        |   －      |
		+------------+-----------+
		| X1         |   IN1     |
		+------------+-----------+
		| X2         |   IN2     |
		+------------+-----------+
		| X3         |   IN3     |
		+------------+-----------+
		| X4         |   IN4     |
		+------------+-----------+

程序示例：
----------

.. code-block:: python

  # main.py -- put your code here!
  import pyb
  from stepmotor import stepping

  def main():
	while True:
	stepping('X1','X2','X3','X4',0,2)

  main()
